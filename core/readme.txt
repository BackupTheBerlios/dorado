Dorado 1.0
http://dorado.planetes.de

Dorado ist ein Installationsprogramm für Tomcat in der Version 5.5.x. 
Der Installer installiert den Tomcat und integriert in direkt ins System, so
dass er über das Dienstepanel oder einem ähnlichen Verwaltungstool direkt 
kontrolliert werden kann.

Darüberhinaus liefert Dorado eine ready-to-use Konfiguration und einen 
optionalen Installer für das JK-Modul für die Anbindung an Apache2 und IIS.
